## 都道府県別総人口推移
 - - -
    
### 概要
RESAS APIから取得した都道府県毎の項目別人口推移をグラフ表示しています。
  
### 技術
Nuxt.js
  
### デプロイ先
[都道府県別総人口推移](https://yumemi-test-7milm21r2-kake-ps-projects.vercel.app/ 'Vercel')
  
### 開発規則
- issue名
`機能(_ファイル名)`

- ブランチ名
`ブランチ名/#issue番号_動詞_機能`
ブランチ名：feature, hotfix
動詞：add, remove, update

- コミットメッセージ
`種類_概要`
種別：feat...新機能、fix...バグ修正、refactor...リファクタ
